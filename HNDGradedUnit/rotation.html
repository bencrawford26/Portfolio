Rotation

This page covers the most notable apps I created when I was working through rotation exercises. While I understand that the concept of rotation isn't particularly interesting - I do however, still feel as if it's a topic worth mentioning due to it's complexity! Rotation on it's own is fairly simple with p5, but when you're trying to realistically simulate the rotation of an objeact that's moving, things become more complicated.

Here is one of the most basic examples of rotation, a simple baton-like object spinning in a circular path. This application was created using only p5.js' in-built rotate function. Angular velocity and acceleration play no part in this example.

Here we have an example of a car moving around an empty space. I know it's quite dull but it's a good example of a user-controlled object with rotation. Control the car using the up, down, left and right arrow keys. One problem with this example is that I've failed to implement a braking system so once the car starts moving it can't stop!

To wrap up rotation, I have an example of the ship I used in my asteroids page without any of the asteroids around to help show how the ship itself works. The ship essentially rotates around an axis in a circular motion, when thrust is applied using the z key the ship moves in the direction it's facing. I also added a small red flash on the ship's thrusters to make it ever so slightly less boring!
