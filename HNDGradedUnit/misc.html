This first app is essentially a gradient generator. In order to generate these gradients, the entire pixel array must be sorted through and coloured accordingly, making this app very heavy on processing. For safety's sake, and so the site doesn't crash, I've set it to only load when the mouse is clicked on the canvas. Have a loof at whate different gradients can be created based on the mouse position.

This app is basically a bubble generator! The idea behind it was to simulate bubbles rising and popping in a glass of fizzy juice! To do this I used the movers from previous examples and gave them erratic movement from left to right with a consistently increasing height. You can add bubbles to the canvas by clicking on it, press any key to remove the last bubble!

This app converts an image to grayscale and then sets the brightness of each pixel relevant to it's distance from the mouse position. the brightest pixels are on the mouses position, the brightness is then reduced towards the centre of the ring which appears around the mouse. Finally, the brightness increases again as the ring widens, leaving all of the pixels outwith the radius pure white. This app is also very heavy on processing so you'll have to click the canvas to use it, too.

Last of all are two very simple apps. The first is essentially a simulation of a smashed monitor. I thought it'd be a good joke to run it fullscreen and convince a friend you've smashed their monitor by accident! The second app uses the same mouse radius effect as the example above but without an image in the background. I just thought it was a nice effect more than anything else so I threw it in here! Both of these apps are also heavy to run in a browser, so I had to make them only run when clicked too.
